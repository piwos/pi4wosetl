Projeto Integrador em Computação IV

Tema escolhido pelo grupo com base no tema norteador da Univesp

Desenvolver um software com framework web ou aplicativo que utilize banco de dados, inclua script web (Javascript), nuvem, acessibilidade, controle de versão, integração contínua e testes. Incluir um dos: uso e fornecimento de API, análises de dados e IoT.

Objetivos:

  • Realizar análise dos dados da pesquisa da WoS
  • Utilização framework web Django
  • Banco de dados PostgreSQL
  • Layout responsivo para todos os dispositivos
  • Análise de dados com as bibliotecas Pandas
  • Criação de gráficos com Google Charts
  • Desenvolvimento com metodologia ágil
